# Summer Camper Site Project

## Overview
This project is a **web application for campers**, developed during summer vacation.  
The platform allows users to browse available camping spots, make reservations, and view campsite details.  

## Features

- User registration and authentication  
- Browse campsites with descriptions and images  
- Make reservations and view booking history  
- Admin panel for managing campsite listings  
- Responsive frontend interface  

---

## Technologies Used

- **Backend:** Django  
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript  
- **Others:** RESTful API, JSON, JWT Authentication  

---

## How to Run

### Backend

```bash
# Navigate to backend folder
cd backend

# Run the server
python app.py
```

### Frontend

```bash
# Navigate to frontend folder
cd frontend

# Install dependencies
npm install

# Run development server
npm start

Open your browser at http://localhost:3000 to use the application.

```
